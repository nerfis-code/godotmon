refactor	Cambios de código sin alterar funcionalidad	    refactor(ui): optimize menu navigation
docs	    Actualización de documentación	                docs: update API reference
assets	    Cambios en arte/sonido	                        assets(sfx): add new explosion sounds
config	    Cambios en configuraciones	                    config: update .gitignore
